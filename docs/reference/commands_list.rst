
.. _list-of-mrtrix3-commands:

########################
List of MRtrix3 commands
########################


.. toctree::
   :maxdepth: 1



   commands/5tt2gmwmi: Generate a mask image appropriate for seeding streamlines on the grey matter - white matter interface

   commands/5tt2vis: Generate an image for visualisation purposes from an ACT 5TT segmented anatomical image

   commands/5ttcheck: Thoroughly check that one or more images conform to the expected ACT five-tissue-type (5TT) format

   commands/5ttedit: Manually set the partial volume fractions in an ACT five-tissue-type (5TT) image using mask images

   commands/afdconnectivity: Obtain an estimate of fibre connectivity between two regions using AFD and streamlines tractography

   commands/amp2response: Estimate response function coefficients based on the DWI signal in single-fibre voxels

   commands/amp2sh: Convert a set of amplitudes (defined along a set of corresponding directions) to their spherical harmonic representation

   commands/connectome2tck: Extract streamlines from a tractogram based on their assignment to parcellated nodes

   commands/connectomestats: Connectome group-wise statistics at the edge level using non-parametric permutation testing

   commands/dcmedit: Edit DICOM file in-place

   commands/dcminfo: Output DICOM fields in human-readable format

   commands/dirflip: Optimise the polarity of the directions in a scheme with respect to a unipolar electrostatic repulsion model, by inversion of individual directions

   commands/dirgen: Generate a set of uniformly distributed directions using a bipolar electrostatic repulsion model

   commands/dirmerge: Splice or merge sets of directions over multiple shells into a single set, in such a way as to maintain near-optimality upon truncation

   commands/dirorder: Reorder a set of directions to ensure near-uniformity upon truncation - i.e. if the scan is terminated early, the acquired directions are still close to optimal

   commands/dirsplit: Split a set of evenly distributed directions (as generated by dirgen) into approximately uniformly distributed subsets

   commands/dirstat: Report statistics on a direction set

   commands/dwi2adc: Convert mean dwi (trace-weighted) images to mean ADC maps

   commands/dwi2fod: Estimate fibre orientation distributions from diffusion data using spherical deconvolution

   commands/dwi2mask: Generates a whole brain mask from a DWI image

   commands/dwi2noise: Estimate noise level voxel-wise using residuals from a truncated SH fit

   commands/dwi2tensor: Diffusion (kurtosis) tensor estimation using iteratively reweighted linear least squares estimator

   commands/dwidenoise: Denoise DWI data and estimate the noise level based on the optimal threshold for PCA

   commands/dwiextract: Extract diffusion-weighted volumes, b=0 volumes, or certain shells from a DWI dataset

   commands/dwinormalise: Intensity normalise the b=0 signal within a supplied white matter mask

   commands/fixel2sh: Convert a fixel-based sparse-data image into an spherical harmonic image

   commands/fixel2tsf: Map fixel values to a track scalar file based on an input tractogram

   commands/fixel2voxel: Convert a fixel-based sparse-data image into some form of scalar image

   commands/fixelcfestats: Fixel-based analysis using connectivity-based fixel enhancement and non-parametric permutation testing

   commands/fixelconvert: Convert between the old format fixel image (*.msf / *.msh) and the new fixel directory format

   commands/fixelcorrespondence: Obtain fixel-fixel correpondence between a subject fixel image and a template fixel mask

   commands/fixelcrop: Crop/remove fixels from sparse fixel image using a binary fixel mask

   commands/fixelreorient: Reorient fixel directions

   commands/fod2dec: Generate FOD-based DEC maps, with optional panchromatic sharpening and/or luminance/perception correction

   commands/fod2fixel: Use a fast-marching level-set method to segment fibre orientation distributions, and save parameters of interest as fixel images

   commands/label2colour: Convert a parcellated image (where values are node indices) into a colour image

   commands/label2mesh: Generate meshes from a label image

   commands/labelconvert: Convert a connectome node image from one lookup table to another

   commands/maskdump: Print out the locations of all non-zero voxels in a mask image

   commands/maskfilter: Perform filtering operations on 3D / 4D mask images

   commands/mesh2pve: Convert a mesh surface to a partial volume estimation image

   commands/meshconvert: Convert meshes between different formats, and apply transformations

   commands/meshfilter: Apply filter operations to meshes

   commands/mraverageheader: Calculate the average (unbiased) coordinate space of all input images

   commands/mrcalc: Apply generic voxel-wise mathematical operations to images

   commands/mrcat: Concatenate several images into one

   commands/mrcheckerboardmask: Create bitwise checkerboard image

   commands/mrclusterstats: Voxel-based analysis using permutation testing and threshold-free cluster enhancement

   commands/mrconvert: Perform conversion between different file types and optionally extract a subset of the input image

   commands/mrcrop: Crop an image to a reduced field of view

   commands/mrdump: Print out the values within an image

   commands/mredit: Directly edit the intensities within an image from the command-line

   commands/mrfilter: Perform filtering operations on 3D / 4D MR images

   commands/mrhistmatch: Modify the intensities of one image to match the histogram of another via a non-linear transform

   commands/mrhistogram: Generate a histogram of image intensities

   commands/mrinfo: Display image header information, or extract specific information from the header

   commands/mrmath: Compute summary statistic on image intensities either across images, or along a specified axis of a single image

   commands/mrmesh: Generate a mesh file from an image

   commands/mrmetric: Computes a dissimilarity metric between two images

   commands/mrmodelfield: Model an input image using low frequency 3D polynomial basis functions

   commands/mrpad: Pad an image to increase the FOV

   commands/mrregister: Register two images together using a rigid, affine or a non-linear transformation model

   commands/mrresize: Resize an image by defining the new image resolution, voxel size or a scale factor

   commands/mrstats: Compute images statistics

   commands/mrthreshold: Create bitwise image by thresholding image intensity

   commands/mrtransform: Apply spatial transformations to an image

   commands/mrview: The MRtrix image viewer

   commands/mtbin: Multi-Tissue Bias field correction and Intensity Normalisation (MTBIN)

   commands/mtnormalise: Multi-tissue normalise

   commands/peaks2amp: Convert peak directions image to amplitudes

   commands/sh2amp: Evaluate the amplitude of an image of spherical harmonic functions along specified directions

   commands/sh2peaks: Extract the peaks of a spherical harmonic function at each voxel, by commencing a Newton search along a set of specified directions

   commands/sh2power: Compute the total power of a spherical harmonics image

   commands/sh2response: Generate an appropriate response function from the image data for spherical deconvolution

   commands/shbasis: Examine the values in spherical harmonic images to estimate (and optionally change) the SH basis used

   commands/shconv: Perform a spherical convolution

   commands/shview: View spherical harmonics surface plots

   commands/tck2connectome: Generate a connectome matrix from a streamlines file and a node parcellation image

   commands/tck2fixel: Compute a fixel TDI map from a tractogram

   commands/tckconvert: Convert between different track file formats

   commands/tckedit: Perform various editing operations on track files

   commands/tckgen: Perform streamlines tractography

   commands/tckglobal: Multi-Shell Multi-Tissue Global Tractography

   commands/tckinfo: Print out information about a track file

   commands/tckmap: Use track data as a form of contrast for producing a high-resolution image

   commands/tcknormalise: Apply a normalisation map to a tracks file

   commands/tckresample: Resample each streamline in a track file to a new set of vertices

   commands/tcksample: Sample values of an associated image along tracks

   commands/tcksift2: Successor to the SIFT method; instead of removing streamlines, use an EM framework to find an appropriate cross-section multiplier for each streamline

   commands/tcksift: Filter a whole-brain fibre-tracking data set such that the streamline densities match the FOD lobe integrals

   commands/tckstats: Calculate statistics on streamlines length

   commands/tensor2metric: Generate maps of tensor-derived parameters

   commands/transformcalc: Perform calculations on linear transformation matrices

   commands/transformcompose: Compose any number of linear transformations and/or warps into a single transformation

   commands/transformconvert: Convert linear transformation matrices

   commands/tsfdivide: Divide corresponding values in track scalar files

   commands/tsfinfo: Print out information about a track scalar file

   commands/tsfmult: Multiply corresponding values in track scalar files

   commands/tsfsmooth: Gaussian filter a track scalar file

   commands/tsfthreshold: Threshold and invert track scalar files

   commands/vectorstats: Statistical testing of vector data using non-parametric permutation testing

   commands/voxel2fixel: Map the scalar value in each voxel to all fixels within that voxel

   commands/warp2metric: Compute fixel-wise or voxel-wise metrics from a 4D deformation field

   commands/warpconvert: Convert between different representations of a non-linear warp

   commands/warpcorrect: Replaces voxels in a deformation field that point to 0,0,0 with nan,nan,nan

   commands/warpinit: Create an initial warp image, representing an identity transformation
