fixup_ANTS_warp

Description

fix up issues with ANTS-generated transformations, whereby locations out of the fielf of view of the original image are assigned nonsense value.

These nonsense values seem to correspond with the values in the very first voxel of the original image (i.e. the voxel at [ 0 0 0 ]). The fix is to read that value from the original image, then identify voxels with the same set of values in the transformed images, and set these locations to NaN.

syntax:     fixup_ANTS_warp [ options ] warp original output

Arguments

warp the input warp image to be fixed up.
original the original warp image pre-normalisation (as generated using gen_unit_warp).
output the corrected output warp image.

Options

-info display information messages.
-quiet do not display information messages or progress status.
-debug display debugging messages.
-help display this information page and exit.
-version display version information and exit.
